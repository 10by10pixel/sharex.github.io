Settings folder

How to change settings folder?

By default ShareX is using "Documents\ShareX" folder.

If you want to change this folder, you need to create a text file with the file name: "PersonalPath.cfg" inside the folder where ShareX.exe resides. A typical example will be: "C:\Program Files\ShareX\PersonalPath.cfg"

Inside this config file you can write which folder you want to use as the settings folder. You can use a relative path. If this config file does not exist or don't contain any text then ShareX will use default folder which is "Documents\ShareX".

How to use ShareX in portable mode?

There is two ways:

  • You can open ShareX with -portable command line argument.
  • You can create "PersonalPath.cfg" file inside the folder where ShareX.exe resides and write "ShareX" which is a relative path, so when ShareX is run it will create "ShareX" named folder to where "PersonalPath.cfg" is and use it as the settings folder.

Both methods are doing exactly the same thing but PersonalPath.cfg is giving more control about where settings folder will be.

What if you don't want use any settings folder at all?

You can open ShareX with -sandbox command line argument so ShareX will open with default settings and won't save any settings. But you still need to disable saving images because it is using Screenshots folder which is changeable from settings window.

Supported command line arguments

-multi or -m allows ShareX to load multiple instances. By default it not possible to use more than one ShareX instance.

-silent or -s loads ShareX to tray without showing the main window.

-portable or -p loads ShareX in portable mode. In portable mode, the application will create a folder (if it doesn't exist already) in its startup path to save configuration files and images.

-clipboardupload performs clipboard upload as soon as ShareX opens.

"filepath" uploads file.

Screen recording with x264 encoder

1. Set hotkey for screen recording.

This hotkey will be used to start recording or stop recording if recording has started already.

2. Click on edit button, go to Screen recorder tab.

In output select "AVI CLI encoder" to use x264 encoder after the video is done recording.

FPS (frame per second) indicates how many frames per second it is recording. If you record at 10 FPS for 5 seconds, then you will have captured 50 frames. The lower the number of frames, the smaller the video size but also the choppier it may look. Setting 10 FPS is my suggestion if you are capturing the entire desktop.

If you check the "Fixed duration" option then you can tell ShareX you want to record for a specified duration. Otherwise, if you want to record as long as you wish you can do so by not checking "Fixed duration" and stop the recording by simply pressing the same hotkey you used to record or by clicking the red tray icon.

Set settings according to this screenshot if you are unsure what to select, these are recommended settings:

3. Open the screen recorder from here:

or you can press the hotkey you had set previously.

You need to select the region where you wish the application to record. Region is up to you. I don't suggest selecting fullscreen region because it can increase video size greatly.

4. Now you need to download x264 encoder.

It is a free and open source encoder and will keep your videos file size down while still making the video look good.

You can download the compiled version from here: http://www.x264.nl/x264_main.php

8bit-depth version with 32bit/64bit depending on your operating system.

I downloaded: 64bit 8bit-depth from mirror 01.

or if you want to compile yourself: http://www.videolan.org/developers/x264.html

After you download it, copy x264.exe to where ShareX is installed (Example: C:\Program Files\ShareX). It is not necessary to copy but if you copy it then you don't need to set encoder path setting because I already set default encoder path to "x264.exe".

5. Now you can start recording.

The region will show a red border for 1 second and then turn to green which means after 1 second the recording will start and you can stop it with hotkey/tray icon any time you want.

After you have stopped it, x264.exe will open in background and encode the recorded video. This x264 window is hidden from you. After encoding has finished ShareX will upload it to the file destination you selected before (in my case it is Dropbox).

Upload screenshots with Steam Overlay UI

Note: You can use the ShareX 'watch folder' feature with the Steam overlay UI to upload screenshots automatically. Below is an instruction for it.

If you're using Steam, you can take a screenshot from a game and upload it automatically using ShareX and the Steam overlay UI.

1. Open Steam settings menu and switch to the 'in-game' tab.

Check "Save an uncompressed copy outside of Steam" and choose a folder to store screenshots by clicking 'select external screenshots folder'.

2. (Optional)

You will probably want to convert the image formats automatically because Steam will save your screenshots uncompressed and those files can be pretty big. The image below shows how you can use ShareX to convert them to a .jpg or .png type in the settings menu.

3. Use the ShareX settings again to create a 'watch folder' like this:

The 'watch folder' will automatically upload new images from the folder you selected as they are added. The watch folder path should point to your directory of uncompressed Steam screenshots.

4. You can now open your Steam game and press your Steam screenshot key (set to F12 by default) at any time to export a screenshot.

ShareX will notice the new file appear in the folder you specified and upload it automatically. If you completed step 2, ShareX will convert the file to a more sensible format before uploading.

Note: This trick also works with any non-steam game in which you have enabled the Steam overlay UI.

How to use Dropbox for image uploads?

1. Destinations -> Image uploader: File Uploader

2. Destinations -> File uploader: Dropbox

3. Destinations -> Outputs configuration...

4. Tab: (File uploaders -> Dropbox) -> Press "1. Open authorize page..." button

5. In your default browser dropbox page will open, press "Allow" button.

6. After see Success page in Dropbox web site turn back to ShareX configuration window and press "2. Complete authorization" button.

7. Now if in "After capture" menu is "Upload image to host" is checked then when you take screenshot with any hotkey (Example: Ctrl + Printscreen) then image will be automatically uploaded to Dropbox.

Note: New Dropbox accounts don't have public folder feature so if you want to use this feature you must enable it with this URL: https://www.dropbox.com/enable_public_folder (https://www.dropbox.com/help/16/en)

Configuring FTP accounts

Automatic folder creation

ShareX can automatically create sub-folders in your FTP Server based on the SubFolderPath settings. FTP Path of a file is shown below: 

URL = ftp:// + Host + : + Port + SubFolderPath + FileName

URL creation based on FTP Account settings

The usual URL creation procedure is as follows: 

URL = HttpHomePath + SubFolderPath + FileName

This default behavior of URL creation can be customized.

If HttpHomePath starts with the @ char then, 

URL = HttpHomePath + FileName

If HttpHomePath is empty then, 

URL = Host + SubFolderPath + FileName

Note: If URL does not start with http:// then it will be added automatically. The / char will be handled automatically; you don't need to add / char to the beginning or end of paths.

If Host starts with ftp. then it will be automatically removed while making URL (when using empty HttpHomePath or when using % in HttpHomePath).

Capturing shapes

All you need to know about capturing shapes with tips and useful hotkeys

Instructions

  • After creating region(s) double click anywhere or press Enter to capture these region(s)
  • You can right click on region to remove it
  • Space key captures full screen
  • Right click empty area or press Esc to cancel capture

When region is not selected or when drawing region

  • Arrow keys will move cursor position
  • Holding Ctrl will increase cursor move speed to 5 pixel instead 1 pixel

When drawing region

  • Holding Shift will use proportional resizing (Width and height will be same size like square or circle)

When region is selected

  • Arrow keys will move left top position for resize region
  • Tab key will swap this anchor position to right bottom
  • Holding Shift will move the entire region instead of resizing
  • Holding Ctrl will increase resize/move speed to 5 pixel instead 1 pixel

Other hotkeys

  • Use mouse wheel to change magnifier pixel count
  • If you hold Ctrl while using mouse wheel then magnifier pixel size will change instead pixel count
  • When using Rounded Rectangle region pressing + or - (Numpad) will change corner radius
  • When using Triangle region pressing + or - (Numpad) will change triangle angle
  • I key toggles coordinate and size information visibility
  • C key toggles screen wide crosshair visibility
  • M key toggles magnifier visibility
  • Q key toggles quick crop
