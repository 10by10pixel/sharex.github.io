Upload screenshots with Steam Overlay UI

Note: You can use the ShareX "watch folder" feature with the Steam overlay UI to upload screenshots automatically. Below is an instruction for it.

If you're using Steam, you can take a screenshot from a game and upload it automatically using ShareX and the Steam overlay UI.

1. Open Steam settings menu and switch to the "in-game" tab.

Check "Save an uncompressed copy" and choose a folder to store screenshots by clicking "Screenshot folder" button.

2. You will probably want to convert the image formats automatically because Steam will save your screenshots uncompressed and those files can be pretty big. The image below shows how you can use ShareX to convert them to a .jpg or .png type in the settings menu.

3. Use the ShareX settings again to create a "watch folder" like this:

The "watch folder" will automatically upload new images from the folder you selected as they are added. The watch folder path should point to your directory of uncompressed Steam screenshots.

4. You can now open your Steam game and press your Steam screenshot key (set to F12 by default) at any time to export a screenshot.

ShareX will notice the new file appear in the folder you specified and upload it automatically. If you completed step 2 then ShareX will compress image before uploading.

Note: This trick also works with any non-steam game in which you have enabled the Steam overlay UI.
