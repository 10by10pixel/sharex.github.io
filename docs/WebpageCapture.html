Webpage capture

ShareX does not have a web capture feature because it will involve rendering pages in background using Internet Explorer. You could, however, use extensions/addons developed for Chrome/Firefox browsers to capture the entire webpage and utilise the Watch Folder feature in ShareX to automatically upload them to your designated image hosting service.

Here is an example using Chrome with the "Awesome Screenshot: Capture & Annotate" extension. It will work exactly same for Firefox etc. with different addons.

1. Download Chrome extension from here: Awesome Screenshot: Capture & Annotate

2. Change image save location from here:

3. Open ShareX task settings window, switch to Watch folders tab, add an existing folder to watch:

4. Click "Capture Entire Page" in extension menu:

After capture, ShareX will automatically upload it.
