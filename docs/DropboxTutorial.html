How to use Dropbox for image uploads?

1. Destinations -> Image uploader: File Uploader

2. Destinations -> File uploader: Dropbox

3. Destinations -> Outputs configuration...

4. Tab: (File uploaders -> Dropbox) -> Press "1. Open authorize page..." button

5. In your default browser dropbox page will open, press "Allow" button.

6. After see Success page in Dropbox web site turn back to ShareX configuration window and press "2. Complete authorization" button.

7. Now if in "After capture" menu is "Upload image to host" is checked then when you take screenshot with any hotkey (Example: Ctrl + Printscreen) then image will be automatically uploaded to Dropbox.

Note: New Dropbox accounts don't have public folder feature so if you want to use this feature you must enable it with this URL: https://www.dropbox.com/enable_public_folder (https://www.dropbox.com/help/16/en)
