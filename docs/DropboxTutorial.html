How to use Dropbox for image uploads?

1. Destinations -> Image uploader: File uploader

2. Destinations -> File uploader: Dropbox

3. Destinations settings...

4. File uploaders -> Dropbox -> Press "1. Open authorize page..." button.

5. In your default browser Dropbox authentication page will open, press "Allow" button.

6. After you see the success page in Dropbox web site, go back to ShareX configuration window and press "2. Complete authorization" button.

7. Now if "After capture" menu, ensure that "Upload image to host" is checked so when you take screenshot with any hotkey (Example: Ctrl + Printscreen), image will be automatically uploaded to Dropbox.

Note: New Dropbox accounts don't have public folder feature so if you want to use this feature you must enable it with this URL: https://www.dropbox.com/enable_public_folder ( https://www.dropbox.com/help/16/en)
